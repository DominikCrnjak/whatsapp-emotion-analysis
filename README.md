# whatsapp-emotion-analysis
Web aplikacija razvijena u Pythonu koristeći Streamlit framework za analizu emocija u WhatsApp razgovorima. Aplikacija omogućuje učitavanje izvoza razgovora u .txt formatu, automatsku pretvorbu u strukturirani CSV zapis, klasifikaciju emocija pomoću GoEmotions i vlastitog BERT modela te vizualizaciju rezultata kroz grafove i usporedbu modela.
